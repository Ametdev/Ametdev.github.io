# Портфолио

Современный сайт-визитка для программиста, созданный на Jekyll и GitHub Pages.

## Как редактировать

### Основная информация

Отредактируйте `_config.yml`:
- `title` - заголовок сайта
- `email` - ваш email
- `description` - описание
- `github_username` - ваш GitHub username
- `telegram_username` - ваш Telegram username

### Главная страница

Отредактируйте `index.html`:
- Секция **Hero** - приветствие и заголовок
- Секция **About** - информация о вас
- Секция **Skills** - ваши навыки (добавьте/удалите карточки по необходимости)
- Секция **Projects** - ваши проекты
- Секция **Contact** - контактная информация

### Добавление проектов

Создайте файлы в папке `_projects/`:

```markdown
---
title: Название проекта
tech: Python • Django • PostgreSQL
description: Краткое описание проекта
---

Полное описание проекта...
```

### Стили

Отредактируйте `_sass/mainsass.scss` для изменения:
- Цветов (переменные в `:root`)
- Шрифтов
- Размеров и отступов

### Цветовая схема

Основные цвета определены в переменных CSS:
- `--primary-color` - основной цвет
- `--secondary-color` - вторичный цвет
- `--text-color` - цвет текста

## Локальная разработка

```bash
bundle install
bundle exec jekyll serve
```

Сайт будет доступен по адресу `http://localhost:4000`

## Деплой

Просто сделайте push в GitHub - GitHub Pages автоматически опубликует изменения.
