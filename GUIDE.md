# Руководство по редактированию сайта

## Быстрый старт

### 1. Обновите личную информацию

Откройте `_config.yml` и измените:

```yaml
title: Ваше Имя
email: your.email@example.com
description: Ваше описание
github_username: ваш_github
telegram_username: ваш_telegram
```

### 2. Отредактируйте главную страницу

Откройте `index.html` и измените:

- **Строка 8**: Ваше имя в заголовке
- **Строка 9**: Ваша специализация
- **Строки 19-21**: Текст "Обо мне"
- **Строки 29-47**: Карточки навыков (добавьте/удалите по необходимости)
- **Строки 55-73**: Карточки проектов
- **Строки 82-86**: Контактные ссылки

### 3. Добавьте свои проекты

Создайте файлы в папке `_projects/` по образцу:

```markdown
---
title: Название проекта
tech: Технологии • через • точку
description: Краткое описание
order: 1
---

## Описание проекта
Подробное описание...

## Технологии
- Список технологий

## Результаты
- Достижения
```

### 4. Измените цветовую схему

Откройте `_sass/mainsass.scss` и измените переменные в `:root`:

```scss
:root {
  --primary-color: #2563eb;      // Основной цвет
  --secondary-color: #1e40af;    // Вторичный цвет
  --text-color: #1f2937;         // Цвет текста
  --text-light: #6b7280;         // Светлый текст
}
```

## Структура проекта

```
├── _config.yml              # Основные настройки сайта
├── index.html               # Главная страница
├── _layouts/
│   ├── default.html         # Базовый шаблон
│   └── project.html         # Шаблон страницы проекта
├── _projects/               # Папка с проектами
│   ├── project-1.md
│   ├── project-2.md
│   └── project-3.md
├── _sass/
│   └── mainsass.scss        # Стили сайта
└── assets/
    └── css/
        └── styles.scss      # Импорт стилей
```

## Добавление новых секций

Чтобы добавить новую секцию на главную страницу, добавьте в `index.html`:

```html
<section id="новая-секция" class="section">
  <div class="container">
    <h2 class="section-title">Заголовок секции</h2>
    <div class="about-content">
      <p>Содержимое секции</p>
    </div>
  </div>
</section>
```

Используйте класс `section-alt` для альтернативного фона:

```html
<section class="section section-alt">
```

## Локальная разработка

```bash
# Установка зависимостей
bundle install

# Запуск локального сервера
bundle exec jekyll serve

# Сайт будет доступен по адресу http://localhost:4000
```

## Публикация

1. Сделайте изменения
2. Закоммитьте их: `git add . && git commit -m "Описание изменений"`
3. Отправьте на GitHub: `git push`
4. GitHub Pages автоматически опубликует изменения через 1-2 минуты

## Советы

- Используйте эмодзи в карточках навыков для визуального разнообразия
- Добавляйте ссылки на GitHub репозитории в описания проектов
- Регулярно обновляйте информацию о проектах
- Используйте качественные скриншоты проектов (можно добавить в папку `assets/images/`)

## Дополнительные возможности

### Добавление изображений

Создайте папку `assets/images/` и добавьте изображения:

```markdown
![Описание]({{ '/assets/images/screenshot.png' | relative_url }})
```

### Добавление блога

Файлы в папке `_posts/` автоматически создают блог. Формат имени: `YYYY-MM-DD-название.md`

### SEO оптимизация

Плагин `jekyll-seo-tag` автоматически добавляет мета-теги. Настройте в `_config.yml`:

```yaml
title: Ваше имя
description: Описание для поисковиков
url: https://yourusername.github.io
```
